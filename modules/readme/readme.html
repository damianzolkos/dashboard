Dashboard is a simple web app interface with modules support.

Pages

Dashboard has a build-in pages system.

Modules

Dashboard uses a modules system to add funcionality. Exery module is rendered in its own box.
Every module is a folder in modules directory. Every module can have a separate JS code.
Module simply is a fragment of HTML code, that is rendered within a defined page.
Every module has a module.json file with config of module: name, unique id, defaultPage, visibility. If page is not specified in config.json, dashboard will render a box in defaultPage. If the page does not exist it will be added.

Notifications

Dashboard have a built-in notifications system.
usage:
newNotification('Test notification.');
